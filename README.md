# Vacature Analyse Dashboard

Een interactief dashboard voor het analyseren van vacatures, gebouwd met Python en Streamlit.

## Over het project

Dit project is ontwikkeld als portfolio-project om vaardigheden in data-analyse, Python en dashboardontwikkeling te laten zien. Het dashboard maakt het mogelijk om vacaturedata eenvoudig te filteren, analyseren en visualiseren.

## Functionaliteiten

- Vacatures filteren op stad en functie
- Salarisgegevens analyseren
- Interactieve grafieken
- Zoekfunctie voor vacatures
- Gebruiksvriendelijke interface

## Gebouwd met

- Python
- Pandas
- Streamlit
- Altair
